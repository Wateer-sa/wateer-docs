# Webhooks

Receive real-time notifications about invoice events.

## Overview

Webhooks allow you to receive HTTP callbacks when events occur in your Wateer account.

## Available Events

- `invoice.created` - New invoice created
- `invoice.viewed` - Customer viewed receipt
- `invoice.failed` - Invoice processing failed

## Setup

Contact [support@wateer.com.sa](mailto:support@wateer.com.sa) to configure webhooks for your account.
